# MKRPremiereImageOutputPlan

Builds a live still/graphic output plan for Premiere Pro.

Use it to return generated graphics or stills as new graphic items, replaced clips, or bin imports.

## Inputs

- `asset_path`
- `image_role`
- `target_name`
- `apply_mode`
- `target_sequence_name` (optional)
- `transport_plan_json` (optional)

## Outputs

- `image_output_plan_json`
- `manifest_line`
- `summary_json`
