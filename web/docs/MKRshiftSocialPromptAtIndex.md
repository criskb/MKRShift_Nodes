# MKRShift Social Prompt At Index

Picks one prompt and negative prompt pair from the JSON arrays generated by the social pack workflow.

## What It Does

- Reads `prompts_json` and `negative_prompts_json`.
- Returns one prompt pair plus the resolved index and total count.
- Supports `Clamp` and `Wrap` indexing so downstream loops can stay simple.

## Typical Use

1. Feed in the prompt arrays from `MKRshiftSocialPackBuilder`.
2. Pick an `index`.
3. Use `Wrap` if you are iterating beyond the array length.
4. Send the chosen prompt strings into generation nodes.

## Notes

- This node is the clean bridge between pack planning and actual prompt execution.
