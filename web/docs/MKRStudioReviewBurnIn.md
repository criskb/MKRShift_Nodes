# MKRStudioReviewBurnIn

Applies a lightweight branded review overlay to an image batch without expanding to a full review-frame canvas.

## Inputs

- `image`: Input image batch.
- `title`, `subtitle`, `badge`: Overlay labels. Leave them blank to inherit from `delivery_plan_json`.
- `theme`: Visual style shared with the other studio nodes.
- `footer_left`, `footer_right`: Bottom metadata fields. These can also be inherited from the delivery plan.
- `inset_px`, `band_height_px`, `accent_width_px`: Burn-in layout controls.
- `opacity`: Overlay panel opacity.
- `show_frame_index`: Appends `FRAME 01`, `FRAME 02`, and so on.
- `delivery_plan_json` (optional): Reuses labels generated by `MKRStudioDeliveryPlan`.

## Outputs

- `image`: Burned-in image batch.
- `burnin_info`: JSON summary with the labels and layout used.

## Use Cases

- Quick client proofs where a full frame wrapper is too heavy.
- Dailies or WIP overlays driven by the same metadata as the delivery plan.
- Batch frame reviews after extraction from a video node.
