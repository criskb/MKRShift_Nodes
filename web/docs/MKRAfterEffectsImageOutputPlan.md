# MKRAfterEffectsImageOutputPlan

Builds a live still/plate output plan for After Effects.

Use it to send generated stills or plate elements back into a comp as imported footage, replaced layer sources, or background plates.

## Inputs

- `asset_path`
- `image_role`
- `target_name`
- `apply_mode`
- `target_comp_name` (optional)
- `transport_plan_json` (optional)

## Outputs

- `image_output_plan_json`
- `manifest_line`
- `summary_json`
