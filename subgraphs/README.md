# MKRShift Subgraph Blueprints

This folder is reserved for real exported ComfyUI subgraph blueprints.

Important:

- ComfyUI expects these files to use the same format as actual exported subgraph/workflow JSON.
- Placeholder JSON files will fail blueprint loading.

For now, this folder is intentionally kept documentation-only until we export valid subgraphs directly from ComfyUI.
