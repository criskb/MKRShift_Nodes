# MKRPublishManifestAtIndex

`MKRPublishManifestAtIndex` pulls one row out of a manifest generated by `MKRPublishAssetManifest`.

## Inputs

- `manifest_json`: Manifest payload.
- `index`: Row to select.
- `index_mode`: Interpret `index` as zero-based list index or display index.

## Outputs

- `row_json`: Selected row as JSON.
- `filename`: Row filename.
- `title`: Row title.
- `alt_text`: Row alt text.
- `tags_csv`: Row tags joined as CSV text.

## Notes

- Handy for driving save paths, captions, or downstream per-asset routing without manually parsing JSON.
