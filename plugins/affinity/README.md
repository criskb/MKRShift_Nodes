# MKRShift Affinity Plugin

Affinity looks most promising through Photoshop-plugin compatibility plus packet-based exchange.

Use together with:

- `MKRAffinityDocumentImport`
- `MKRAffinityExportPlan`
- `MKRAffinityPhotoshopPluginPlan`

This folder is install-facing documentation and a future landing point for Affinity-specific compatibility packaging.
