# x1LightWrapComposite

`x1LightWrapComposite` composites a foreground over a background using a matte and adds practical edge light-wrap from the background back into the subject.

Use it for:

- plate integration
- CG-over-live-action comp
- quick beauty comp passes

Outputs:

- composited `IMAGE`
- wrap `MASK`
- info `STRING`
