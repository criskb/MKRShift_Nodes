# MKRShift Blender Plugin

Use the packaged Blender add-on zip:

- `/Users/crisbjorndal/ComfyUI/custom_nodes/MKRShift_Nodes/blender_extension/dist/mkrshift_blender_bridge.zip`

Install it from Blender preferences:

- `Edit > Preferences > Add-ons > Install...`

This folder exists so the plugin installer can present Blender alongside the other host integrations.
