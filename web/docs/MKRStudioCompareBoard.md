# MKRStudioCompareBoard

Builds a branded A/B comparison board from two image inputs for export-ready review.

## Inputs

- `image_a`, `image_b`: The two image batches to compare. The node pairs them by batch index and uses the shorter batch if counts differ.
- `title`, `subtitle`: Header text. Leave blank to inherit from `delivery_plan_json`.
- `label_a`, `label_b`: Panel labels such as `Before` and `After`.
- `theme`: Visual style shared with the studio review nodes.
- `orientation`: Horizontal or vertical layout.
- `footer_left`, `footer_right`: Footer text, optionally inherited from the delivery plan.
- `margin_px`, `gutter_px`, `header_px`, `footer_px`: Layout controls.
- `shadow_strength`: Card shadow depth.
- `show_index`: Appends the pair index to the footer.
- `delivery_plan_json` (optional): Reuses labels generated by `MKRStudioDeliveryPlan`.

## Outputs

- `image`: The comparison-board image batch.
- `compare_info`: JSON metadata describing the chosen layout.

## Use Cases

- Before/after grade reviews.
- Paintover or comp note exports.
- Client-facing comparisons that need branding and version context.
