# MKRTouchDesignerFramePlan

`MKRTouchDesignerFramePlan` builds a return packet for TouchDesigner-side bridge components.

Typical uses:

- file-based TOP ingest
- Spout / NDI routing plans
- operator-path + TOP-name manifest generation

Outputs:

- `td_frame_plan_json`
- `manifest_line`
- `summary_json`
