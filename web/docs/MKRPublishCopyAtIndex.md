# MKRPublishCopyAtIndex

`MKRPublishCopyAtIndex` extracts one caption/copy variant from the JSON deck generated by `MKRPublishCopyDeck`.

## Inputs

- `deck_json`: Copy deck payload.
- `index`: Zero-based variant index.

## Outputs

- `variant_json`: Selected variant row as JSON.
- `headline`: Variant headline.
- `caption`: Full composed caption.
- `hashtags_csv`: Variant hashtags joined as CSV text.

## Notes

- Useful when you want one specific caption variant to feed into save metadata, overlays, or external publishing steps.
